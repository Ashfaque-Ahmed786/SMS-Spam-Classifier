# SMS / Email Spam Classifier

This is a **Machine Learning project** that classifies SMS or email messages as **Spam** or **Not Spam** using Python, Scikit-learn, and Streamlit.

## Features
- Text preprocessing using NLTK (tokenization, stopwords removal, stemming)
- TF-IDF vectorization
- Multinomial Naive Bayes classifier
- Interactive Streamlit web app
